Deberás desarrollar una aplicación web compuesta por las siguientes páginas:

index.jsp=> se trata de la home page y contendrá dos vínculos (etiquetas html <a>…</a>) cuyos textos sensible
serán:
Tabla multiplicar
Circunferencia
tabla.jsp=> por medio de un formulario permite introducir un número que será enviado al servlet Tabla que
generará una página dinámica para mostrar la tabla de multiplicar del número introducido.
Tabla.java: servlet requerido desde la página tabla.jsp para mostrar la tabla de multiplicar del número introducido
por el usuario.
circunferencia.jsp=> por medio de un formulario permite introducir un número que será enviado a la página
longarea.jsp.
longarea.jsp=> ofrece como resultado la longitud y el área de la circunferencia cuyo radio fue introducido por el
usuario.
En todas las páginas del sitio web deberá mostrarse el identificador de la sesión.
