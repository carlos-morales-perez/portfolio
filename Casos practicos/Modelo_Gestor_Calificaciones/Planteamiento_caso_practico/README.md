El siguiente caso práctico servirá para evaluar tus conocimientos tanto en diseño web como en desarrollo en JavaScript. Deberás desarrollar un sitio web de acuerdo a las siguientes especificaciones.
- La web servirá para consultar las calificaciones de varios alumnos en distintos cursos.
- Deberás elaborar tu trabajo utilizando tres ficheros independientes: archivo .html, archivo .css y archivo .js.
- Dentro de tu archivo .js deberás declarar una variable llamada alumnos que contendrá un array de 4 objetos JSON (4 alumnos).
- Para cada objeto JSON se guardará el nombre del alumno, el curso que está realizando y un array de calificaciones (asignatura y nota).
- Tu página web debe permitir al usuario seleccionar un alumno por medio de una lista desplegable (elemento HTML <select>).
- Cuando el usuario selecciona un alumno, la página mostrará todos los datos de dicho alumno: curso que está realizando y calificaciones.
- También deberá mostrarse la nota media del alumno seleccionado.
- El diseño de la página web es libre.
Consideraciones, para la parte de diseño se valorará:
- En qué grado has realizado un mínimo trabajo de maquetación y no simplemente colocar los elementos uno detrás de otro como caigan.
- La utilización de etiquetas semánticas de HTML5 (<header>, <main>, <footer>, etc.).
- La inclusión de elementos que faciliten el posicionamiento SEO, la usabilidad y la accesibilidad.
- Un mínimo diseño estético aplicando atributos CSS.
